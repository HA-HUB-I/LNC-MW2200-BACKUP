# LNC MW2200A Modbus Web Dashboard (PWA)

Това приложение предоставя в реално време мониторинг и управление на CNC машина LNC (Тайван).

## 🚀 Текущ Статус (26 Април 2026)
- **Backend:** Python Flask + Pymodbus.
- **Интерфейс:** Мобилно-оптимизиран (PWA), поддържа MM/CM превключване.
- **Unit ID:** 0 (Работи най-стабилно с диагностичните регистри).

## 📊 Потвърдени Данни
- **Живи Координати:** X (R12065), Y (R12038), Z (R12033).
- **Actual позиции:** X (R12032), Y (R12038).
- **Шпиндел:** R1007 (RPM).
- **Скорост:** R11638 (mm/min).
- **Цикъл:** Софтуерно изчисляване чрез R0 бит 2.

## 🛠 Управление
- Поддържа `START`, `HOLD`, `RESET`, `STOP`.
- M-кодове за **Вакуум (M10/M11)**.
- Активни аларми при включен шпиндел или активни стопери.

## 🔄 Как се рестартира (Чист старт)
Ако промените не се виждат, използвайте:
```powershell
.\modbus_web\nuclear_restart.ps1
```
Този скрипт убива старите процеси, чисти кеша и пуска приложението на чисто.

## 📂 Структура на откритията
Всички нови регистри се тестват първо в `modbus_web/discovery/` чрез:
- `live_logger.py`: Логер с превод на човешки език.
- `super_live_scanner.py`: Скенер за високочестотни промени.
